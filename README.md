# react-native-pdf-thumbnail

Generate thumbnail for PDF files

## Installation

```sh
npm install react-native-pdf-thumbnail
```

## Usage

```js
import PdfThumbnail from "react-native-pdf-thumbnail";

// ...

const result = await PdfThumbnail.multiply(3, 7);
```
